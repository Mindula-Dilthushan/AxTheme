# Ax-Theme
</div>

## Installation

Install this package using npm:

```
npm install ax-theme
```

## Usage

Import the `AxTheme` object in your React.js application:

```javascript
import AxTheme from 'ax-theme';
```

```javascript
const Welcome = () => {
    return(
        <View style={[AxTheme.bgRed]}>
        </View>
    )
}
exports default Welcome;
```
